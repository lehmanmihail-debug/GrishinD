у каких групп какие дисциплины будут, по сколько часов отводится на лекции/семинары/зачеты/экзамены на каждый семестр).

5. База Данных - это структурированная коллекция данных (информации),

Реляционные MySQL PostgreSQL
Нереляционные mongodb

РЕЛЯЦИОННАЯ База данных - хранят информацию в табличном виде, позволяют связывать таблицы между собой, более строгая БД
в ядре Зашит язык SQL который позволяет гибко управлять структурами и таблицами в ней, и выполнять различные операции с данными
НЕРЕЛЯЦИОННАЯ - Работают на основе ключей значения 

В Реляционных Базах данных должны удовлетворять правила нормальных форм(НФ)

1НФ - Описывает атомарность(неделимость) каждого атрибута, отсутствие одинаковых колонок и наличию свойству уникальности каждой строки 
2НФ - Первичный ключ в одной таблице должен быть один 
Это удаление первичных составных ключей 
 Таблица находится Все не ключевые столбцы зависят 

Атрибут - 
Таблица базы данных - это сущность = модель = таблицами
У Сущности есть Атрибуты - это название колонки, что идет внутри колонки это 

Для первичных ключей ID тип serial
