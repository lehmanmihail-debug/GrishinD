Метод Push
Добавление элемента в конец масива
let array = [-3, 0, 3.14, "string", true, "22" ]
array.push(10)
console.log(array)
    в конце массива добавляется числовая 10

console.log(array.length) вывод количества элементов в массиве

Метод Shift
Удаляет элемент с индексем 0 тоесть первый элемент в массиве

array.shift()

Метод unshift
Добавляет элемент в начало массива 

array.unshift("hello world")

Метод forEach 

Применяет функцию к нашему Массиву

Выводит положительные элементы:
const array = [-3,-1, 0, 3.14, 10, 20 ]

array.forEach(element => {
    if (element >= 0) {
        console.log(element)
    }
    {
        return false
    }
});

console.log(array)
---
Применяет функцию для каждого элемента массива (не создовая нового массива)
---
умножает на два каждый элемент
const array = [-3,-1, 0, 3.14, 10, 20 ]

array.forEach(element => {
    element = element * 2
    console.log(element)
});
console.log(array)
---

Чаще используется Метод map (но этот метод создает новый массив)

const array = [-3,-1, 0, 3.14, 10, 20 ]

const newArray = array.map(element =>{
    return element*2
}) 
console.log(newArray)

console.log(newArray.includes("string")) это поиск элемента в массиве (true / false)
